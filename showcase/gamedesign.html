Mad Libs

User enters words to create a funny story.



What problem did the project solve?

This project solved the problem of paper Mad Libs which are wasteful and single-use. Paper Mad Libs must be thrown away after one use. Also, they can only be enjoyed by one person. By virtualizing Mad Libs, the same prompt can be enjoyed by many people and no paper is wasted.



Technologies used:

Scratch



Reflection:

This project helped me use "join" blocks, user inputted answers, and variables in Scratch to create a fun story. Next time, I would improve the animations in the story to be more condusive with the storyline.

Create Performance Task for AP Computer Science Principles

The AP Computer Science Principles Create performance task is part of the AP Exam. Students will be provided at least 9 hours in class to complete the performance task. The Create performance task focuses on the creation of a computer program and students will develop a Personalized Project Reference that contains their computer code. On the end-of-course exam, students will respond to two questions related to the code contained in their Personalized Project Reference.
